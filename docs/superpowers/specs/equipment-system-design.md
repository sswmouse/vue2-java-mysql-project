## 一、概述

本文档描述DNF 115级重力之泉版本装备系统的数据结构和前端展示方案。

### 1.1 装备体系

- 115级套装装备共有12套
- 每套包含11件装备
- 装备分为多个品级，品级越高属性越强
- 通过套装积分触发不同阶段的套装效果

### 1.2 装备部位

| 分类 | 部位 |
|-----|------|
| 防具 | 头肩、胸甲、腰带、下装、鞋 |
| 首饰 | 项链、手镯、戒指 |
| 特殊装备 | 辅助装备、魔法石、耳环 |

---

## 二、装备品级

### 2.1 品级分类

| 部位 | 品级（从低到高） |
|-----|----------------|
| 防具 | 稀有 → 神器 → 传说 → **史诗** |
| 首饰 | 稀有 → 神器 → 传说 → 史诗 → **太初** + **黑牙史诗** + **黑牙太初** |
| 特殊装备 | 稀有 → 神器 → 传说 → **史诗** |

> **注意**：
> - 防具和特殊装备最高品级为史诗，不支持太初/黑牙
> - 首饰最高品级为太初，支持黑牙史诗和黑牙太初

### 2.2 品级说明

| 品级 | 说明 |
|-----|------|
| 稀有 | 基础品级，无套装属性但有积分 |
| 神器 | 进阶品级 |
| 传说 | 高级品级 |
| 史诗 | 顶级品级（防具/特殊）/ 高阶品级（首饰） |
| 太初 | 最高品级，仅首饰支持 |
| 黑牙史诗 | 首饰专属，比史诗积分低但名望高 |
| 黑牙太初 | 首饰专属，比太初积分低但名望高 |

## 三、套装积分系统

### 3.1 单件装备积分

| 品级 | 积分 |
|-----|-----|
| 稀有 | 65 |
| 神器 | 115 |
| 传说 | 165 |
| 史诗 | 215 |
| 太初 | 265 |
| 黑牙史诗 | 165 |
| 黑牙太初 | 215 |

> **黑牙积分规则**：黑牙品级积分比原品级低50（黑牙史诗=165，黑牙太初=215）

### 3.2 单件装备名望

| 品级 | 名望 |
|-----|-----|
| 稀有 | 1000 |
| 神器 | 1100 |
| 传说 | 1200 |
| 史诗 | 1300 |
| 太初 | 1400 |
| 黑牙史诗 | 1500 |
| 黑牙太初 | 1600 |

> **名望计算**：稀有1000为基础，每提升一级+100；黑牙品级名望比原品级高200

### 3.3 套装阶段触发

共有21个阶段，从低到高排列：

| 阶段 | 触发积分 | 阶段额外名望 | 累计名望 |
|-----|---------|-------------|---------|
| 稀有1 | 750 | 0 | 0 |
| 稀有2 | 835 | +1000 | 1000 |
| 稀有3 | 920 | +2000 | 2000 |
| 稀有4 | 1005 | +3000 | 3000 |
| 稀有5 | 1090 | +4000 | 4000 |
| 神器1 | 1200 | +5500 | 5500 |
| 神器2 | 1285 | +6500 | 6500 |
| 神器3 | 1370 | +7500 | 7500 |
| 神器4 | 1455 | +8500 | 8500 |
| 神器5 | 1540 | +9500 | 9500 |
| 传说1 | 1650 | +11000 | 11000 |
| 传说2 | 1735 | +12000 | 12000 |
| 传说3 | 1820 | +13000 | 13000 |
| 传说4 | 1905 | +14000 | 14000 |
| 传说5 | 1990 | +15000 | 15000 |
| 史诗1 | 2100 | +16500 | 16500 |
| 史诗2 | 2185 | +17500 | 17500 |
| 史诗3 | 2270 | +18500 | 18500 |
| 史诗4 | 2355 | +19500 | 19500 |
| 史诗5 | 2440 | +20500 | 20500 |
| 太初 | 2550 | +22000 | 22000 |

> **积分计算规则**：
> - 太初积分：2550
> - 每下降一个小段位（同一品级内）减85分
> - 每下降一个大段位（跨品级）减110分
>
> **名望计算规则**：
> - 每个小阶段额外+1000名望
> - 每个大阶段额外+2500名望（跨品级时额外加成）

## 四、数据库设计

### 4.1 数据表结构

#### 4.1.1 装备套装表 (equipment_sets)

| 字段 | 类型 | 说明 |
|-----|------|------|
| id | BIGINT | 主键 |
| set_name | VARCHAR(100) | 套装名称 |
| set_index | INT | 套装序号(1-12) |
| set_type | VARCHAR(20) | 套装类型：normal/special |
| description | TEXT | 套装描述 |
| tech_attack_bonus | FLOAT | 技攻加成 |
| attack_bonus | INT | 攻击强化加成 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

#### 4.1.2 装备详情表 (equipment_items)

| 字段 | 类型 | 说明 |
|-----|------|------|
| id | BIGINT | 主键 |
| set_id | BIGINT | 所属套装ID |
| slot_type | VARCHAR(20) | 部位类型：armor/jewelry/special |
| slot_name | VARCHAR(50) | 部位名称：头肩/胸甲/腰带等 |
| item_name | VARCHAR(100) | 装备名称 |
| rarity_stats | JSON | 各品级属性(JSON格式) |
| rarity_points | JSON | 各品级积分(JSON格式) |
| rarity_fame | JSON | 各品级名望(JSON格式) |
| sort_order | INT | 排序 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

> **rarity_stats JSON格式**：
```json
{
  "rare": { "str": 50, "int": 50, "crit": 5 },
  "legendary": { "str": 80, "int": 80, "crit": 8 },
  "epic": { "str": 120, "int": 120, "crit": 12 },
  "mythic": { "str": 160, "int": 160, "crit": 16 }
}
```

#### 4.1.3 套装效果表 (set_effects)

| 字段 | 类型 | 说明 |
|-----|------|------|
| id | BIGINT | 主键 |
| set_id | BIGINT | 所属套装ID |
| stage | VARCHAR(20) | 阶段：rare1/rare2/.../mythic |
| threshold | INT | 触发积分阈值 |
| fame_bonus | INT | 阶段额外名望 |
| effects | JSON | 套装效果描述 |
| created_at | DATETIME | 创建时间 |

#### 4.1.4 阶段配置表 (stage_configs)

| 字段 | 类型 | 说明 |
|-----|------|------|
| id | BIGINT | 主键 |
| stage | VARCHAR(20) | 阶段名称 |
| stage_order | INT | 阶段顺序(1-21) |
| threshold | INT | 触发积分 |
| fame_bonus | INT | 额外名望 |
| stage_type | VARCHAR(20) | 阶段类型：minor/major |

## 六、API设计

### 6.1 获取装备套装列表

```
GET /api/equipment-sets
Response: [
  { id, setName, setIndex, currentPoints, currentStage, ... }
]
```

### 6.2 获取套装详情

```
GET /api/equipment-sets/:id
Response: {
  setInfo: { ... },
  items: [ ... ],
  effects: [ ... ]
}
```

### 6.3 获取单件装备详情

```
GET /api/equipment-items/:id?rarity=epic
Response: {
  itemInfo: { ... },
  stats: { ... },
  points: 215,
  fame: 1300
}
```
